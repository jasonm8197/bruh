File | Edit | View | Tools | Help

File

New
Edit a new caveblock. All data from the current block will be erased.

Open
Load an existing caveblock from storage.

Save
Save current caveblock. If it's untitled it will work like save as.

Save As ...
Save current caveblock and prompt for filename and location. The .CAV extension is used for Digging Jim caveblocks.

Settings ...
Various general variables for the builder. Here you can set the number of objects used in random distribution and toggle the animation on/off.

Test Cave
Test the current level. This feature is only availible in the registered version.


Edit

Undo
Undo the last change to the map. Insert/delete operation cannot be undone.

Copy Level
Copy the current level to the clipboard.

Paste Level
If the clipboard is not empty the level residing there will be pasted on the current level and thus overwriting it.

Clear Level
Clear current level.

Insert Level
Inserts an cave after the current. This will increase the number of levels with one and move all subsequent levels forward.

Remove Level
Deletes the current levels and moves all subsequent levels backwards. This will decrease the number of levels with one.

Cave Properties ...
Allows you to edit the properties for the current cave. These includes :

    - Diamonds to collect
    - Diamond value
    - Extra diamond value
    - Amoeba growth maximum (in blocks)
    - Time
    - Magic wall expire time
    - Amoeba and plasma growth speed
    - Cave color settings(Hue, Saturation, Lightness)

Random Distribution
Places a number of the current object randomly in the cave. The amount can be changed under settings in the file menu.

View

Next Level
Views the next level. You can also use the plus-key for faster access to this function.

Previous Level
Views the previous level. You can also use the minus-key.

Normal Blocks
This will select the 32x32 object size also used in the actual game.

Small Blocks
When using the smaller objectsize you can survey a bigger portion of the level. When the editor-window is maximized you should be able to edit almost the entire level(depending on your screen resolution) at once without having to scroll.

Tools

Lets you select the object to draw with. A reference of the objects can be fould in the enviroment section. It's much easier to select the objects on the toolboard beside the editfield.

Help

Contents
General help for Builder.

Object reference
Help on the objects. It will bring you the to enviroment section.
