The Builder Editfield

The editfield is where the actual designing of the cave takes place. You can navigate the cave with scrollbars on the sides, the arrowkeys or alternatively with the minimap. If you want to survey a bigger portion of the cave while editing you can choose "small blocks" in the view pulldown menu. Pressing the left mouse button places the selected object at the mousecursor position(you can read the coordinates on the right side under the toolboard). Holding down the button while moving the mouse will allow you to cover larger areas with the same object. Right-clicking on the editfield will bring up the tool context menu, but the easiest way to select an object is using the toolboard. The size of a cave is 50x30 objects and is always lined by solid walls.
